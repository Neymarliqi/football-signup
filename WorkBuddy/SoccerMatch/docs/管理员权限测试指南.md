## 常见问题排查

### 问题1：首页不显示「⚙ 管理」按钮

**原因分析：**
- `app.globalData.isAdmin` 可能还是 `false`
- 管理员检查可能还在进行中（异步操作）

**排查步骤：**
1. 在控制台检查：`getApp().globalData.isAdmin`
2. 如果是 `false`，检查 `admins` 集合是否有对应记录
3. 手动触发管理员检查：
   ```javascript
   getApp().checkAdmin('o1hdR4zFTRIhXtek3ck_1rNqqS4g')
   ```

### 问题2：控制台提示权限错误

**原因分析：**
- `admins` 集合的权限设置可能不对

**解决方案：**
在云开发控制台 → 数据库 → `admins` 集合 → 权限设置，改为：
- 所有用户：**可读**
- 所有用户：**可写**（或者设置为「仅创建者可写」）

### 问题3：无法进入管理后台

**原因分析：**
- `admin.js` 的权限检查触发了
- `app.globalData.isAdmin` 不是 `true`

**解决方案：**
1. 确认 `admins` 集合有记录
2. 确认 openid 完全匹配（包括大小写）
3. 在控制台手动设置：
   ```javascript
   getApp().globalData.isAdmin = true
   ```
