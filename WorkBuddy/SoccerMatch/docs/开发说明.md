# 约球助手小程序 - 开发说明

## 📋 项目概述

一个功能完整的足球约球小程序，基于微信云开发，无需独立服务器。用户可以查看活动、报名/待定/请假、查看战术板，管理员可以发布活动和管理。

---

## 🎯 核心功能

### 用户端
- **首页**：活动列表、公告滚动、报名人数进度、球员头像展示
- **活动详情**：报名/待定/请假、腾讯地图导航、战术板入口
- **战术板**：可视化足球场、5 种阵型预设、拖拽球员位置
- **个人中心**：头像昵称设置、位置偏好、参与记录

### 管理端
- **发布活动**：时间、地点（地图选点）、人数、费用、公告
- **管理后台**：活动管理、公告发布、球员列表

---

## 🏗 技术栈

| 技术 | 说明 |
|------|------|
| 前端 | 微信小程序原生框架（WXML + WXSS + JS） |

在云开发控制台 -> 数据库，创建以下集合：
1. **activities** - 活动表
2. **announcements** - 公告表
3. **users** - 用户表
4. **tactics** - 战术表
5. **admins** - 管理员表

### 第七步：设置数据库权限
在云开发控制台 -> 数据库 -> 权限设置：
- **activities**：
  ```
  {
    "read": true,
    "write": "doc.createdBy == auth.openid"
  }
  ```
- **announcements / tactics / admins**：
  ```
  {
    "read": true,
    "write": "auth.openid != null"
  }
  ```

### 第八步：添加管理员
在 `admins` 集合中手动添加一条记录：
```json
{
  "openid": "你的微信openid"
}
```
**获取 openid 方法**：在小程序启动后，打开调试控制台，输入 `wx.getStorageSync('openid')` 复制显示的值。

### 第九步：测试运行
1. 在 `activities` 集合中添加一条测试活动
2. 在开发者工具点击编译，查看首页
3. 测试报名、地图导航、战术板等功能

## 📍 地图功能说明

### 选点（发布活动）
- 使用 `wx.chooseLocation` 调起微信内置地图
- 返回经纬度（gcj02 坐标系）+ 地址信息
- 保存到数据库供导航使用

### 导航（活动详情）
- 使用 `wx.openLocation` 调起腾讯地图导航
- 需要经纬度和地址信息
- **不需要额外申请腾讯地图 API Key**

## 📸 头像昵称获取

### 微信最新组件
- 头像：`<button open-type="chooseAvatar" bind:chooseavatar="onChooseAvatar">`
- 昵称：`<input type="nickname" placeholder="请输入昵称">`

### 上传流程
1. 用户选择头像 → 获取临时路径
2. 上传到云存储：`wx.cloud.uploadFile`
3. 获取 fileID → 保存到数据库 `users` 集合
