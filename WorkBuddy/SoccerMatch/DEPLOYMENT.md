### 1.3 隐私保护指引（审核必填）

#### 1.3.1 后台配置

**位置**：微信公众平台 → 设置 → 基本设置 → 用户隐私保护指引

**填写内容**：

```
一、我们如何收集和使用您的信息

1. 为帮助您注册和使用本小程序，我们可能会收集您的微信昵称、头像、OpenID。

2. 为提供活动位置服务，我们可能会收集您的地理位置信息（仅在您选择地点时使用）。

3. 为记录您的活动报名情况，我们会存储您报名的活动信息。

二、我们如何存储和保护您的信息

您的信息存储在微信云开发服务中，我们采用业界标准的安全措施保护您的数据。

三、您的权利

您有权查看、修改或删除您的个人信息。如有需要，请联系开发者。

四、联系我们

如您对本隐私保护指引有任何疑问，请通过小程序内的反馈功能联系我们。
```

#### 1.3.2 接口勾选

在隐私指引设置页面，勾选以下接口：

| 接口 | 用途 |
|------|------|
| `getUserProfile` 或 `getUserInfo` | 获取用户昵称、头像 |
| `getLocation` | 地图导航功能 |
| `chooseLocation` | 腾讯地图选点插件 |
| `getOpenid` | 用户身份识别 |

#### 1.3.3 用户生成内容场景声明

**位置**：微信公众平台 → 设置 → 基本设置 → 用户生成内容场景声明

| 选项 | 选择 |

| 用户生成内容 | ✅ **包含** |

## 五、常见问题与解决方案

### 5.1 地理位置权限提示

**问题**：提交审核时提示「地理位置权限」

**解决方案**：
- 点击「申请权限」
- 填写申请理由：
  ```
  本小程序为足球活动报名管理工具，用户报名后需要导航到活动场地，
  因此需要获取用户当前位置用于计算路线和距离。
  ```

### 5.2 隐私指引未完善

**问题**：审核失败，提示「隐私保护指引未完善」

**解决方案**：
- 检查「用户隐私保护指引」是否已填写
- 检查「用户生成内容场景声明」是否已声明
- 检查接口勾选是否完整

### 5.3 服务类目不符

**问题**：审核失败，提示「服务类目与实际功能不符」

**解决方案**：
- 确保选择的是「体育 > 体育场馆服务」或「体育 > 赛事活动」
- 不要选择「工具」或「生活服务」等其他类目

### 5.4 腾讯地图 Key 无效

**问题**：地图选点无法使用

**解决方案**：
- 检查 Key 是否正确：`SXGBZ-RHQ6M-26V6Z-6UTTU-JGKUV-TVFJS`
- 检查 WebServiceAPI 是否开启
- 检查 SN 签名校验是否关闭

### 5.5 云函数未部署

**问题**：调用云函数报错

**解决方案**：
- 打开开发者工具
- 右键 `cloudfunctions` 文件夹 → 「创建并部署：云端安装依赖」
- 等待部署完成（约 1 分钟）

### 5.6 数据库权限问题

**问题**：读取/写入数据库报错

**解决方案**：
- 进入云开发控制台 → 数据库
- 找到对应集合 → 权限设置
- 设置为：「所有用户可读，仅创建者可写」

### 5.7 TabBar 不显示

**问题**：底部导航栏不显示

**解决方案**：
- 检查 `app.json` 中 `tabBar` 配置是否正确
- 检查 `custom-tab-bar` 目录是否存在
- 检查页面 `onShow` 是否同步选中状态

## 六、版本更新流程

### 6.1 版本号规则

```
主版本.次版本.修订号

示例：
1.0.0 - 初始版本
1.1.0 - 新增功能
1.1.1 - Bug 修复
1.2.0 - 重要更新
2.0.0 - 重大版本升级
```

### 6.2 更新步骤

1. 修改代码
2. 更新 `CHANGELOG.md`
3. 提交到 Git
4. 开发者工具上传
5. 体验版测试
6. 提交审核

### 6.3 版本回滚

如果新版本有严重 Bug：

```
微信公众平台 → 版本管理 → 审核版本 → 撤回审核
→ 或回滚到上一个版本
```
