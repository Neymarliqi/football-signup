## 核心功能

### 用户端
- 首页活动列表 + 公告滚动
- 活动详情：报名/待定/请假、地图导航、战术板
- 战术板：可视化足球场、5 种阵型、拖拽球员
- 个人中心：头像昵称、位置偏好

### 管理端
- 发布活动：时间地点地图选点、人数费用
- 管理后台：活动管理、公告发布、球员列表

## 云函数

- `getOpenid`：获取用户 openid
- `updateRegistration`：报名/待定/请假逻辑
- `cancelRegistration`：取消报名

## 重要技术决策

### 地图功能
- 使用微信内置地图（基于腾讯地图）
- 选点：`wx.chooseLocation`
- 导航：`wx.openLocation`
- **不需要额外申请腾讯地图 API Key**
- 坐标系：gcj02（国测局坐标系）

### 头像昵称
- 使用微信最新组件：`<button open-type="chooseAvatar">` 和 `<input type="nickname">`
- 头像上传到云存储，fileID 保存到数据库
- 首次报名时弹窗引导填写
