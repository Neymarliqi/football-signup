### 3. 问题排查与解决
- **Git 锁文件问题**：之前有 `git add -A` 进程僵死 17 分钟，导致 `.git/index.lock` 锁文件存在，新 Git 命令被阻塞
  - 解决方法：`kill -9` 杀掉僵死进程，`rm ~/.git/index.lock` 删除锁文件
- **文件夹重命名识别**：Git 默认把跨文件夹移动识别为"删除+新增"，需要 `git add` 重新识别
