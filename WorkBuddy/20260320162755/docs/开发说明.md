# 约球助手小程序 - 开发说明

## 📋 项目概述

一个功能完整的足球约球小程序，基于微信云开发，无需独立服务器。用户可以查看活动、报名/待定/请假、查看战术板，管理员可以发布活动和管理。

---

## 🎯 核心功能

### 用户端
- **首页**：活动列表、公告滚动、报名人数进度、球员头像展示
- **活动详情**：报名/待定/请假、腾讯地图导航、战术板入口
- **战术板**：可视化足球场、5 种阵型预设、拖拽球员位置
- **个人中心**：头像昵称设置、位置偏好、参与记录

### 管理端
- **发布活动**：时间、地点（地图选点）、人数、费用、公告
- **管理后台**：活动管理、公告发布、球员列表

---

## 🏗 技术栈

| 技术 | 说明 |
|------|------|
| 前端 | 微信小程序原生框架（WXML + WXSS + JS） |
| 后端 | 微信云开发（云函数 + 云数据库 + 云存储） |
| 地图 | 微信内置地图（基于腾讯地图，无需额外 Key） |

## 🗄 数据库设计

### activities（活动集合）
| 字段 | 类型 | 说明 |
|------|------|------|
| title | string | 活动标题 |
| activityDate | date | 踢球日期 |
| time | string | 时间段（如 "19:00 - 21:00"） |
| location | string | 详细地址 |
| locationName | string | 场地名称 |
| latitude | number | 纬度（地图导航） |
| longitude | number | 经度（地图导航） |
| matchType | string | 赛制（11人制/7人制/5人制等） |
| maxPlayers | number | 人数上限 |
| registrations | array | 报名列表（含 status: confirmed/pending/leave） |
| status | string | 状态（open/ongoing/finished/cancelled） |

### announcements（公告集合）
| 字段 | 类型 | 说明 |
|------|------|------|
| content | string | 公告内容 |
| active | boolean | 是否启用 |
| createdAt | date | 创建时间 |

### users（用户集合）
| 字段 | 类型 | 说明 |
|------|------|------|
| nickName | string | 昵称 |
| avatarUrl | string | 头像（云存储 fileID 或 URL） |
| position | string | 位置偏好 |
| updatedAt | date | 更新时间 |

### tactics（战术集合）
| 字段 | 类型 | 说明 |
|------|------|------|
| activityId | string | 关联活动 ID |
| formation | string | 阵型（4-3-3 等） |
| positions | array | 球员位置列表 |

### admins（管理员集合）
| 字段 | 类型 | 说明 |
|------|------|------|
| openid | string | 管理员 openid |

## 🚀 部署步骤

### 1. 微信开发者工具导入项目
1. 打开微信开发者工具
2. 项目目录选择：`/Users/Neymar/WorkBuddy/20260320162755`
3. AppID 填你的小程序 AppID

### 2. 开通云开发
1. 顶部点击「云开发」
2. 开通环境，填写环境名称（如 `football`）
3. 记下环境 ID

### 3. 配置环境 ID
打开 `miniprogram/app.js`，修改：
```js
wx.cloud.init({
  env: 'cloud1-5gbn5i1p97239e9d',  // 你的环境 ID
  traceUser: true,
})
```

### 4. 上传云函数
右键每个云函数文件夹 → 「上传并部署：云端安装依赖」：
- `cloudfunctions/getOpenid`
- `cloudfunctions/updateRegistration`
- `cloudfunctions/cancelRegistration`

### 5. 创建数据库集合
在云开发控制台创建 5 个集合：
- activities（活动）
- announcements（公告）
- users（用户）
- tactics（战术）
- admins（管理员）

权限设为「所有用户可读，仅创建者可写」

### 6. 添加管理员
在 `admins` 集合添加一条记录：
```json
{
  "openid": "你的管理员的 openid"
}
```

## 📍 地图功能说明

### 选点（发布活动）
- 使用 `wx.chooseLocation` 调起微信内置地图
- 返回经纬度（gcj02 坐标系）+ 地址信息
- 保存到数据库供导航使用

### 导航（活动详情）
- 使用 `wx.openLocation` 调起腾讯地图导航
- 需要经纬度和地址信息
- **不需要额外申请腾讯地图 API Key**

## 📸 头像昵称获取

### 微信最新组件
- 头像：`<button open-type="chooseAvatar" bind:chooseavatar="onChooseAvatar">`
- 昵称：`<input type="nickname" placeholder="请输入昵称">`

### 上传流程
1. 用户选择头像 → 获取临时路径
2. 上传到云存储：`wx.cloud.uploadFile`
3. 获取 fileID → 保存到数据库 `users` 集合
