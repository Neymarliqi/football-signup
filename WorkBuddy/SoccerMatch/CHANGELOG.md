# 约球助手更新日志

## v1.1.3 (2026-03-25)

### 战术板页面优化

#### 多人报名场景支持
- **替补区域可滚动**：使用 `scroll-view`，支持40+人报名场景
- **保存按钮固定底部**：始终可见，不会被长列表推到下面
- **统计信息显示**：分隔线显示"已选 X / 总人数"
- **渐变背景优化**：底部按钮区添加渐变，防止文字重叠

#### 球员显示样式统一
- **头像尺寸**：60rpx × 60rpx（场上球员和替补球员统一）
- **昵称显示**：最多8个字符，字体大小15rpx，最大宽度100rpx
- **位置标签**：字体大小10rpx
- **替补球员排列**：行间距24rpx，列间距40rpx（紧凑不拥挤）

#### 拖拽体验优化
- **精确跟随手指/光标**：元素左上角直接对齐触摸位置，无任何偏移
- **性能优化**：移除复杂的尺寸计算，直接使用触摸坐标

## v1.1.2 (2026-03-25)

### 云函数新增
- **deleteActivity**：通过云函数删除活动，绕过客户端权限限制
- **setAdmin**：设置/取消管理员权限

### 管理员权限设计
- 使用单独的 `admins` 集合存储管理员 openid
- `app.js` 中的 `checkAdmin` 函数查询 `admins` 集合判断权限
- 存储完整微信 openid（非短ID）

### 微信头像与昵称获取修复
- 修复个人中心点击头像/昵称时，选择弹窗显示不正确的问题
- 改用 `app.globalData.userInfo` 或 `wx.getStorageSync('userInfo')` 获取已保存的用户信息

## v1.1.1 (2026-03-23)

### 隐私保护
- 新增隐私保护指引页面（`pages/privacy/privacy`）
- 个人中心增加「设置」菜单，包含隐私保护指引入口
- 优化隐私页面样式，采用卡片式布局

### 数据更新体验优化
- **移除下拉刷新**：首页和个人中心关闭下拉刷新
- **智能刷新策略**：
  - 首页：30秒间隔判断 + 数据库实时监听（最近7天活动）
  - 活动详情：精确监听单个活动变化
  - 个人中心：本地缓存优先，后台静默同步
- **性能优化**：减少不必要的数据库查询

## v1.1.0 (2026-03-22)

### UI/UX 全面优化
- **视觉层次分明**：主操作 → 功能操作 → 分隔线 → 管理操作
- **管理操作弱化**：灰色文字、无背景、无icon
- **左对齐原则**：所有操作元素左对齐
- **统一间距**：推荐 12rpx gap

### 首页优化
- 卡片按钮区：主操作→功能操作→分隔线→管理操作
- 删除悬浮发布按钮（TabBar已有）
- 修复取消活动功能（阻止事件冒泡+详细权限检查）

### 活动创建优化
- **赛制类型**：非必填，默认"友谊赛"，支持自定义
- **导航地址**：非必填，调用地图选择位置
- **最大人数**：支持"不限人数"切换
- **报名截止时间**：预设选项（1天/6小时/1小时/0.5小时/不限制）
- **自定义赛制**：每个活动独立存储

### 活动详情优化
- 一步获取用户信息（wx.getUserProfile）
- 自动完成报名
- 简化"待定/请假"逻辑
- 去除确认弹窗

### 自定义 TabBar
- 纯 CSS 绘制图标（房子、人像、加号）
- 未选中：灰色(#999)
- 选中：微信绿(#07c160)，填充浅绿色背景
- 发布按钮：绿色圆形背景 + 白色加号
- 修复导航切换问题（parseInt索引 + 避免重复切换）
