## 🔧 部署前需要配置的内容

### 1. 腾讯位置服务 Key（必需）

地图选点插件需要一个腾讯位置服务 Key，请按以下步骤申请：

#### 步骤 1：注册腾讯位置服务
1. 访问 [腾讯位置服务官网](https://lbs.qq.com/)
2. 使用微信扫码登录
3. 进入控制台

#### 步骤 2：创建应用
1. 点击左侧菜单「应用管理」→「我的应用」
2. 点击「创建应用」
3. 填写应用名称：约球助手
4. 选择应用类型：小程序

#### 步骤 3：添加 Key
1. 在应用详情页点击「添加Key」
2. 填写 Key 名称：约球助手-地图选点
3. 选择「微信小程序」平台
4. 填写小程序 AppID（在微信公众平台获取）
5. 勾选「WebServiceAPI」权限
6. 点击提交

#### 步骤 4：配置到代码
打开 `miniprogram/pages/activity/create.js`，找到第 412 行：

```javascript
// 腾讯位置服务 Key（需要在腾讯位置服务平台申请）
const key = 'YOUR_TENCENT_MAP_KEY' // 请替换为您的Key
```

将 `YOUR_TENCENT_MAP_KEY` 替换为您申请的 Key。

---

### 2. 微信小程序后台配置

#### 添加地图选点插件
1. 登录 [微信公众平台](https://mp.weixin.qq.com/)
2. 进入「设置」→「第三方服务」→「插件管理」
3. 点击「添加插件」
4. 搜索：`腾讯位置服务地图选点`
5. 点击「添加」

#### 配置服务器域名（如使用云开发可跳过）
如果使用云开发，无需配置。如果自建后端，需要配置：
- request 合法域名
- uploadFile 合法域名
- downloadFile 合法域名

#### 配置位置权限
在 `app.json` 中已配置：
```json
"permission": {
  "scope.userLocation": {
    "desc": "获取您的位置用于显示踢球地点附近信息"
  }
}
```

---

### 3. 云开发环境配置

#### 确保云开发环境已启用
1. 在微信开发者工具中点击「云开发」按钮
2. 确保数据库、存储、云函数都已启用

#### 数据库集合
确保以下集合已创建：
- `activities` - 活动数据
- `users` - 用户数据
- `admins` - 管理员数据

#### 云函数
确保以下云函数已部署：
- `getOpenid` - 获取用户 openid
- `updateActivity` - 更新活动
- `updateRegistration` - 更新报名状态
- `cancelRegistration` - 取消报名
- `updateUserPosition` - 更新用户位置

## 🚀 部署到小程序商城的步骤

### 步骤 1：上传代码
1. 打开微信开发者工具
2. 点击右上角「上传」按钮
3. 填写版本号（如：1.2.6）
4. 填写项目备注
5. 点击「上传」

### 步骤 2：提交审核
1. 登录 [微信公众平台](https://mp.weixin.qq.com/)
2. 进入「管理」→「版本管理」
3. 在「开发版本」中找到刚上传的版本
4. 点击「提交审核」
5. 填写审核信息：
   - 功能页面：pages/index/index
   - 功能描述：足球约球活动发布与报名小程序
   - 测试账号：如有需要请提供

### 步骤 3：等待审核
- 审核通常需要 1-3 个工作日
- 审核结果会通过邮件和公众号通知

### 步骤 4：发布上线
1. 审核通过后，在「版本管理」中点击「发布」
2. 确认发布信息
3. 点击「确认发布」
4. 小程序正式上线！

## ⚠️ 重要提醒

### 关于腾讯位置服务 Key
- **免费额度**：腾讯位置服务提供一定的免费调用额度
- **超出收费**：如果用户量较大，可能需要付费
- **Key 安全**：Key 是敏感信息，不要泄露给他人

### 关于微信用户头像昵称
- **隐私合规**：使用用户头像昵称需要遵守微信隐私规范
- **用户授权**：必须获得用户明确授权才能获取
- **数据存储**：用户头像 URL 可能有时效性，建议定期更新

### 关于分享功能
- **朋友圈分享**：需要小程序支持「分享到朋友圈」能力
- **分享图片**：建议配置分享图片，提升点击率
- **分享文案**：分享标题要吸引人，包含关键信息
