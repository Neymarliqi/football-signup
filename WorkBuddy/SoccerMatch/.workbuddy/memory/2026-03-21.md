### 腾讯地图调用说明
- 小程序使用 `wx.openLocation` 调起微信内置地图（基于腾讯地图）
- 不需要额外申请腾讯地图 API Key
- 坐标系要求：gcj02（国测局坐标系），`wx.chooseLocation` 返回的经纬度已符合
